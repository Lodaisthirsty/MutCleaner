# Saving Cleaned Data and Cleaning Artifacts

## Overview

This guide explains how to save the outputs generated by MutCleaner after running a dataset cleaner.

MutCleaner commonly produces two types of outputs:

- **Cleaned dataset**: the final standardized and quality-controlled mutation dataset, used for downstream analysis.
- **Cleaning artifacts**: auxiliary outputs generated during the cleaning process, such as filtered records, discarded rows, intermediate tables, or quality-control results.

The cleaned dataset can be saved directly in the MutCleaner format and reloaded later. Cleaning artifacts are saved as a pickle file by default. Since pickle files are binary and not convenient to inspect directly, users can load the artifact file in Python and export each artifact table to CSV for manual review.

## Main idea

- **Main dataset** → use `dataset.save(path)` → stored in the **mutcleaner format** (default `save_type="mutcleaner"`), which can be reloaded directly by mutcleaner.
- **Filtered / discarded / intermediate rows** (“artifacts”) → use `cleaning_pipeline.save_artifacts(path)`  → saved as a single **pickle** file (`.pkl`) containing multiple DataFrames. → since pickle is binary and not convenient to view in VS Code, we **recommend reading it back in Python and exporting individual tables to CSV**.

---

## Common Workflow

The typical workflow is:

1. Run the cleaner and save the main cleaned dataset.
2. Save cleaning artifacts generated during the cleaning process.
3. Export artifact tables to CSV files.
4. Manually inspect filtered records to verify whether they are truly dirty data.

### Save the main cleaned dataset

```python
# Default: save_type="mutcleaner"
dataset.save("path/to/output_dir")
```

### Save cleaning artifacts and export artifacts to CSV

```python
import pickle
from pathlib import Path

artifact_path = Path("path/to/artifacts.pkl")
artifact_csv_dir = Path("path/to/target_dir")

cleaning_pipeline.save_artifacts(artifact_path)
artifact_csv_dir.mkdir(parents=True, exist_ok=True)

with artifact_path.open("rb") as file:
    artifacts = pickle.load(file)
    
for artifact_name, artifact_df in artifacts.items():
    artifact_df.to_csv(artifact_csv_dir / f"{artifact_name}.csv", index=False)
```

## Example

The following example uses the Human Domainome dataset.

**save the main cleaned data**

```python
hd_dataset.save("path/to/output_dir")
```
**save the cleaning artifacts**

```python
import pickle
from pathlib import Path

artifact_path = Path("path/to/artifacts.pkl")
artifact_csv_dir = Path("path/to/target_dir")

hd_cleaning_pipeline.save_artifacts(artifact_path)
artifact_csv_dir.mkdir(parents=True, exist_ok=True)

with artifact_path.open("rb") as file:
    artifacts = pickle.load(file)
    
for artifact_name, artifact_df in artifacts.items():
    artifact_df.to_csv(artifact_csv_dir / f"{artifact_name}.csv", index=False)
```

## Notes

The exported CSV files are mainly used for manual inspection and audit. For example, users can check whether filtered records are truly problematic before deciding whether the cleaning rules are appropriate.
